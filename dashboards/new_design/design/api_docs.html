Entity Resolution API

The Entity Resolution API allows institutional clients to programmatically query our 1.2 billion-node ontology to identify wallet owners, associated risk scores, and known counterparties.

GET /api/v2/entity/{address}

Parameters

NameTypeDescription
addressstringThe EVM, BTC, or Solana address to query.
include_txbooleanInclude recent transaction hashes.

Example Response

{
  "status": "success",
  "data": {
    "address": "0x3b5d17e2...",
    "entity_name": "Lazarus Group",
    "classification": "THREAT_ACTOR",
    "risk_score": 98.5,
    "sanctions": {
      "ofac_sdn": true,
      "date_added": "2024-05-12T00:00:00Z"
    }
  }
}
